## 📌 Summary
Welcome to the Simple Checkout App!
-----------------------------------
A grocery checkout app doesn't get simpler than this. Wanna see what a little JavaScript can do for your site then give this a try!<br>
<br>
The purpose of this program is to simulate a simple grocery check out as a web application. Just fill out the fields, click "Display Receipt", and watch the magic! 

---

## ⭐ How It Works
1. 🧾 User enters:
   - Customer name  
   - Item name  
   - Price  
   - Quantity  
   - Cash amount  
2. ✅ Input is validated for accuracy and completeness  
3. 🧮 Program calculates:
   - Subtotal  
   - Discount (if applicable)  
   - Tax  
   - Final total  
   - Change due  
4. 📄 A formatted receipt is generated  
5. ⚡ Receipt is displayed via a browser alert  

---

## ✨ Features
- 🛒 Simulates a **real-world checkout process**  
- ✅ **Robust input validation** with user-friendly error messages  
- 💰 Applies **discount logic** for qualifying purchases  
- 🧮 Calculates **tax, totals, and change dynamically**  
- 🧾 Generates a **clean, formatted receipt output**  
- ⚡ Interactive UI using **event-driven JavaScript**
